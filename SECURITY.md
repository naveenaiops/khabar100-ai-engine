# Security Policy

## Supported Versions

We actively support and apply security patches to the following versions of Khabar100:

| Version | Supported |
| ------- | --------- |
| 2.0.x   | Yes       |
| < 2.0.0 | No        |

## Reporting a Vulnerability

We take the security of our services, AI pipelines, and student users very seriously. If you discover any security vulnerabilities, credential leaks, or potential exploits, please do NOT create a public issue on GitHub.

Instead, please report the vulnerability privately to the project maintainers via email at `security@khabar100.in` (simulated address).

### Our Process

- Upon receiving a report, we will investigate and acknowledge the vulnerability within 48 hours.
- We will work closely with you to design a fix or security patch.
- We aim to deploy fixes to production instances within 7 days of confirmation.
- Once a fix is verified, we will publish a security advisory and credit you for the discovery.
